Nombre del proyecto: Inventario Básico
Gestion de inventario para la evaluacion nº 3 Programación Básica

Descrpcion del proyecto: 
Este programa desarrollado en python entrega una interfaz de con solo simple para realizar la gestion de un inventario.
Permitiendo al usuario agregar nuevos productos, ver la lista de los productos, buscar productos especificos y eliminar productos del inventario.

Requisitos:
Version de Python se recomienda usar Python 3.6 o superior
Librerias adicionales no se requiere instalar, ya que solo utiliza funcionalidades basicas o estandar de Python.
